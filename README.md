# QR Code Generator

A professional, high-performance web application designed for generating dynamic QR codes with advanced customization options. Built with a focus on aesthetics and user experience, the application features a premium glassmorphic interface and a custom particle-based animation system.

## Key Features

- **Dynamic QR Generation**: Support for multiple data types including plain text, URLs, WiFi credentials, VCards (contacts), and email addresses.
- **Advanced Customization**: Real-time control over QR code foreground and background colors, margins, and drop-shadow effects.
- **Interactive UI**:
    - **Particle Gathering Animation**: A custom-built animation where geometric shards converge from the screen edges to "assemble" the QR code.
    - **Crunch-and-Expand Button States**: Fluid transitions for the generation button, providing clear state feedback.
    - **Interactive Feedback**: Tactile animations including container shakes and checkmark indicators for action confirmations.
- **One-Click Actions**: Built-in functionality for both downloading the generated image and copying it directly to the system clipboard.

## Technology Stack

- **Frontend**: Standard HTML5, CSS3, and Vanilla JavaScript.
- **Design System**: Glassmorphism with a focus on modern typography and smooth transitions.
- **API Integration**: Utilizes the QRServer API for high-resolution code generation.

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. No build steps or external dependencies required beyond an active internet connection for the QR API.
