# QR cody

just a simple qr code generator i made with html css and vanilla js.
it has some cool animations and you can customize colors and stuff.

## features
- make qr codes for text, urls, wifi, etc
- change colors and margins
- download or copy the qr code
- cool particle animation when generating

## how to run
just download the repo and open `index.html` in your browser. no install needed.

or check the [demo](https://qr.bgtulk.dev/)
